FLOW A — GIÁ & MARKUP (Alphatrans)

  • Alphatrans discount 40% giá Express.
  • Luồng interAsia: chỉ markup 40% → giá bán = rate × 1.4 + $48 (giá trucking).
  • Luồng EU, US: markup 50–55% → giá bán = rate × 1.5–1.55 + $48 (giá trucking).
  • Cùng hãng (ví dụ cùng FedEx) → có thể gom chung 1 truck.
  • Khác hãng (DHL / UPS) → mỗi hãng phải đi riêng 1 truck, vì mã kho ngoại quan khác nhau.

Pre-inbound — ASN Label Generator

Tạo label ASN trực tiếp trong trình duyệt (không gửi data ra ngoài). "ASN Single" = mỗi EAN 1 label (đúng format mẫu hiện tại). "ASN Mix" = label tổng hợp theo carton (nhiều SKU/label) — tự động shrink chữ hoặc phóng to trang khi số SKU trong 1 carton vượt ngưỡng. "PDF In Order" = gộp nhiều PDF label (Single + Mix, hoặc bất kỳ) thành 1 file, tự sắp xếp lại theo Carton No. và cảnh báo nếu thiếu/trùng số — không cần tạo Single/Mix tách riêng nữa, có thể gộp chung vào đây.

📎 Template mẫu (Single + Mix): file Excel mẫu, tô sẵn màu — Đỏ = bắt buộc (MUST), Xanh lá = không bắt buộc (OPTIONAL), Xanh dương = tự động tính (AUTO). Chỉ cần mã EAN/Item code — SKU có thể để trống.
⬇ Tải Template ASN (Single + Mix).xlsx

1. Upload file Excel (sheet "Single")

Cột cố định: B=SKU (không bắt buộc), C=EAN/Item code (BẮT BUỘC), H=Qty/box (bắt buộc), I=Carton qty (tự động = K−J+1), J=Carton start (bắt buộc), K=Carton end, L=MIX marker (không bắt buộc). Dữ liệu bắt đầu từ dòng 5.

Chưa upload file.

2. Tạo label

1. Upload file Excel (sheet "Mix")

Cột cố định: C=EAN/Item code (BẮT BUỘC, dùng làm mã hiển thị trên label), B=SKU (không dùng, có thể để trống), H=Qty (bắt buộc), I=group marker "1"/"MIX" (bắt buộc ở dòng đầu carton), J/K=Carton start/end (bắt buộc), P=Carton ID base (bắt buộc). Carton ID in ra = cột P + "-" + tổng Qty của carton.

Chưa upload file.


2. Tạo label

Gộp 1 hoặc nhiều file PDF label đã tạo (vd: file ASN Single + file ASN Mix, hoặc bất kỳ PDF label nào cùng format) thành 1 PDF duy nhất, tự động sắp xếp lại toàn bộ trang theo Carton No. tăng dần (đọc trực tiếp từ dòng chữ "Carton No." in trên mỗi trang) — đồng thời cảnh báo nếu có carton bị thiếu số hoặc trùng/chồng số. Không cần phân biệt file nào là Single hay Mix.

1. Upload PDF (chọn được nhiều file cùng lúc)

Mỗi trang PDF phải có dòng chữ "Carton No. <số>" (đúng format label hiện tại của Single/Mix). Chọn nhiều file cùng lúc để gộp — hệ thống sẽ đọc hết rồi sắp xếp chung theo Carton No.

Chưa chọn file.
